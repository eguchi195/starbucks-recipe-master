# スタバレシピマスター ☕

スターバックス新人パートナー向けの、レシピ最速暗記PWA。
ドリンクコード・手順番号・サイズ別数値（S/T/G/V）をスマホで効率よく覚えられます。

## 機能

- **レシピ図鑑**: ドリンク名 or ドリンクコードで高速検索（ひらがな入力OK）
- **付箋スクラッチ**: サイズ別数値をタップでペリッとめくって暗記チェック（開閉状態は保存される）
- **ホット/アイス切り替え**: `[I]` 工程はアイス時のみ表示、ホット専用工程はSKIPバッジ
- **暗記ドリル**: コードマスター（名前⇄コード4択）＋ 数値ドリル（サイズ・温度・工程名から数値4択）
- **お気に入り**: ハートで登録（localStorage保存）
- **PWA**: オフライン対応。スマホのホーム画面に追加して使えます

## 起動方法

```bash
npm install
npm run dev        # 開発サーバー
npm run build      # 本番ビルド（dist/ 生成）
npm run preview    # ビルド結果の確認（PWA/Service Workerはこちらで動作）
```

## バナー画像の差し替え

`src/assets/banner.png`（jpg / jpeg / webp も可）を置くだけで、
詳細ページ最上部のSVGプレースホルダーが自動で実画像に差し替わります。

## データの編集

レシピはすべて `src/data/drinks.ts` に定義されています。

- `code: null` … ドリンクコード未設定（UIでは「—」表示）
- `temp: 'hot' | 'ice' | 'both'` … 工程の温度区分（`ice` = 原文の `[I]` 工程）
- `isCustom: true` … 「シロップ（希）」＝カスタム時のみ追加（※カスタムバッジ）
- 手順番号が飛んでいる箇所は詰めずに欠番のまま（UIが「（このステップなし）」を補完）
- 例外: トリプルエスプレッソラテ(3EL)は `availableSizes: ['T']`、キッズココアは `availableSizes: []`
